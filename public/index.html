Launch Readiness

What's included

Mood tracking, journaling, guided exercises, persistent data storage, and Gemini-powered support chat with mental-health-only prompt guardrails.

How the key works

Your Gemini API key stays on the server in .env as GEMINI_API_KEY. It is not stored in the browser UI.

Deploy anywhere

Use npm start on Render, Railway, Fly.io, or a VPS. Set the environment variables in your host dashboard.

Safety stance

The chatbot avoids unrelated topics, does not diagnose, and escalates immediate-risk language to emergency guidance.
